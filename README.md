# TechCon_HTML
This project is a basic multi-page website built to practice HTML structure, semantic tags, and accessible design.
The website represents a fictional technology conference called TechCon 2024.

# Pages Included

index.html – Homepage introducing the conference and featuring a promotional video

about.html – About page with history, mission, and past speakers

schedule.html – Event schedule presented in a structured table

register.html – Registration page with a user-friendly form

contact.html – Contact page with email link, social links, map, and message form

Key HTML Concepts Practiced

Semantic structure (header, main, footer, nav, section, article)

Embedding media (video and Google Maps iframe)

Creating tables with captions and headers

Building accessible forms

Using alt text and labels for accessibility

Creating internal links between pages

# Purpose

This project helps strengthen foundational HTML skills and demonstrates how to structure a small, multi-page website using clean, semantic, and accessible code.
